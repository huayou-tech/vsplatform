# 【监控组件】DeviceStatuService - 设备状态监控服务

## 项目简介

DeviceStatuService是华又科技视频监控平台的设备状态监控核心组件，专门负责监控前端设备的在线状态、心跳检测和异常告警等功能。该服务采用高效的异步IOCP模型，能够稳定监控数千台设备的状态变化。

**GitHub源码获取**：[点击下载DeviceStatuService完整工程](https://github.com/huayoutech/videomonitor-platform)

## 核心功能特性

### 🔍 设备状态监控

1. **实时状态检测**
   - 心跳包定时发送和接收
   - 设备在线/离线状态精准判断
   - 连接异常自动检测和恢复
   - 状态变化实时通知机制

2. **健康度评估**
   - 设备响应时间统计分析

3. **告警管理**
   - 多级别告警机制
   - 告警去重和合并
   - 告警历史记录
   - 告警通知推送

struct HeartbeatInfo
{

    bool isAwaitingResponse;                            // 等待响应标志

};
