# CSDN 博客每日自动发布 - 快速开始指南

## 🎯 功能介绍

这是一个专为华又科技视频监控平台技术文章设计的自动发布系统，可以**每天自动选择一篇文章**并发布到 CSDN 博客。

### ✨ 核心特性

- ✅ **智能轮询**: 自动扫描所有文章，按顺序每天发布一篇
- ✅ **状态跟踪**: 记录已发布文章，避免重复
- ✅ **内容优化**: 自动添加项目介绍、GitHub 链接、互动引导
- ✅ **一键发布**: 简单操作即可完成发布准备
- ✅ **循环发布**: 所有文章发完后自动重新开始

## 🚀 三种使用方式

### 方式一：每日自动脚本 (最简单) ⭐推荐

适合：想要快速完成发布的用户

**操作步骤:**

```bash
# 双击运行或在命令行执行
python auto_publish_daily.py
```

**效果:**
- 自动选择一篇未发布的文章
- 生成优化后的内容文件
- 更新发布日志
- 告诉您下一步操作

**示例输出:**
```

✅ 找到 20 篇文章
✅ 选中：Windows IOCP异步模型深度解析

✅ 发布日志已更新 (总计：1 篇)
🎉 今日发布任务完成!
```

## 📝 完整发布流程

### 第一步：运行自动发布工具

```bash
python auto_publish_daily.py
```

### 第二步：获取准备好的内容

工具会生成一个文件：`ready_to_publish_YYYYMMDD_HHMMSS.md`

例如：`ready_to_publish_20260303_221149.md`

### 第三步：登录 CSDN 发布

1. 打开 CSDN 写文章页面：https://mp.csdn.net/mp_blog/write/article
2. 复制生成的 markdown 内容
3. 粘贴到编辑器
4. 设置分类和标签
5. 点击发布

### 第四步：推广

发布后:
- 分享到技术群、朋友圈
- 回复评论互动
- 在 GitHub 项目 README 中添加文章链接

## ⏰ 设置每日定时发布

### 方法 1: Windows 任务计划程序

1. 打开 **任务计划程序** (Win + R → taskschd.msc)
2. 点击 **创建基本任务**
3. 填写信息:
   - 名称：`CSDN 博客自动发布`
   - 触发器：**每天**
   - 时间：**09:00** (或您想要的时间)
   - 操作：**启动程序**
   - 程序：`python.exe`
   - 参数：`auto_publish_daily.py`
   - 起始于：`E:\华又科技\VSS\VSPlatform`
4. 完成

### 方法 2: 使用批处理 + 任务计划

创建 `daily_publish.bat`:

```batch
@echo off
cd /d E:\华又科技\VSS\VSPlatform
python auto_publish_daily.py
```

然后在任务计划程序中设置每天运行此批处理文件。

### 方法 3: 持续运行的自动发布

```bash
python csdn_auto_publisher.py
# 选择选项 3: 启动每日自动发布
```

这会持续运行并在每天指定时间自动发布。

## ❓ 常见问题

### Q1: 如何知道今天发布了哪篇文章？

查看 `publish_log.json` 中的 `last_publish_date` 和最后一条记录。

### Q2: 发完所有文章后怎么办？

系统会自动重新开始循环，继续从第一篇开始发布。

### Q3: 可以手动指定发布某篇文章吗？

可以，使用交互式工具 `csdn_auto_publisher.py`，选择相应功能。

### Q4: 能否同时发布到多个平台？

目前只支持 CSDN，但可以扩展代码支持知乎、掘金等平台。
